Detailed Description

A weighted collection item.

Represents a value retrieved from the dataset together with a weight indicating the proportion or confidence of this value (out of 65535^2). Used for profile-level weighting (from profile groups) where the weight indicates the proportion of the matched IP range attributable to this profile, or for value-level weighting stored in the Value record, or both.
