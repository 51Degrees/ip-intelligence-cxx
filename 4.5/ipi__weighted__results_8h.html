ipi_weighted_results.h File Reference

Detailed Description

Defines structures and functions for handling weighted values in IP Intelligence results.

This file provides the data structures and functions needed to work with weighted values of different types (int, double, bool, byte, string) in the IP Intelligence system. Weighted values include both the actual value and a weighting that indicates the confidence or importance of that value.
