performanceState Struct Reference

Detailed Description

Shared performance state.

Single state containing everything needed for running and then reporting the performance tests.

All members are immutable except runningThreads which is updated via an interlocked compare and exchange.

Examples
IpIntelligence/PerfIpi.c, and IpIntelligence/Performance.c.
