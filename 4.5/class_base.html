Base Class Reference

Detailed Description

Base test class used to carry out memory leak checks.

All test classes extending this will be subject to checks ensuring that all memory allocated during the test is cleared up. This class also adds the ability to check that the expected amount of memory is allocated by setting the expected allocation.
